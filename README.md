# KEEWS
P2P地震情報を利用して  
Minecraftサーバーで地震速報をリアルタイムに受け取ることができるプラグイン

## 機能
◆震度速報/震源に関する情報/震源・震度情報  
◆緊急地震速報[警報] (EEW)   
◆津波情報   
がグローバルチャットに自動的に表示   

管理者用  
◇リアルタイム接続とサンドボックス接続の切り替え  
◇特定の震源地の情報を非表示・特定震度以上の表示設定  

## コマンド
`/keews help` ヘルプを表示  
`/keews info` 接続情報表示   
`/keews reconnect` 手動で再接続  
2時間おきに自動再接続されます。  
`/keews switch <realtime/sandbox>` ﾘｱﾙﾀｲﾑ接続かｻﾝﾄﾞﾎﾞｯｸｽ環境に切り替え  
`/keews setting sindo` 震度表示の設定状況を表示  
`/keews setting sindo <震源地名> <震度> <over/hidden>` 設定した震源地名の表示設定  
  over:設定した震源地で該当する震度以上を表示します。  
  hidden:該当する地震を非表示にします。  
  震度の数値：1,2,3,4,5-,5+,6-,6+,7  
  削除する場合はフォルダ内sindo_setting.txtを参照  

## 緊急地震速報 (警報) EEW について
震度5弱以上の揺れが予想される場合、EEW(警報)の第1報目の情報を配信します。  
震度5弱以上の揺れが予想される地域内が拡大されるごとに追加で配信されます。(1回ずつ）
